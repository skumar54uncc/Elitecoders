# Logo Folder

Please add your logo files here.

## Recommended formats:
- `logo.svg` - Vector format (preferred)
- `logo.png` - PNG with transparent background
- `logo-dark.png` - Dark version for light backgrounds
- `logo-light.png` - Light version for dark backgrounds

## Recommended sizes:
- Main logo: 200px width (or SVG)
- Favicon: 32x32px or 16x16px

## Usage:
The logo will be used in:
- Header navigation (left side)
- Footer (optional)
- Browser favicon
